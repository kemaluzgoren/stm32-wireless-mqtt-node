# STM32 NUCLEO-F439ZI MQTT Publish & Subscribe Project

## Project Description

This project aims to exchange data with MQTT protocol over Ethernet using STM32 NUCLEO-F439ZI development board with lwIP. Mosquitto, as a MQTT broker, is run on the computer.

The project performs data publish and subscribe functions using MQTT, a lightweight messaging protocol used in IoT (Internet of Things) applications.

The STM32 connects to the MQTT broker via Ethernet connection, publishes sensor data and processes incoming data by subscribing from specified topics.
